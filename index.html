Deferred linking test

The button below copies a token to your clipboard and redirects you to the company's mobile app if it is installed. If the app is not installed it should redirect you to the app store.
The token starts with "company:" so when pasting the content the app knows if it's a valid token or not discarding it if it's not.

If the clipboard content is a a valid token the app submits it to the server to be validated and eventually login the user.
